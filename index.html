üöÄ How to Use

As Owner:

  1. Login with owner@club.com / owner123
  2. View complete dashboard with all statistics
  3. Navigate to Manage Events to add/edit/delete events
  4. Go to Manage Admins to add or remove admin users
  5. Use Customize to change club name and description
  6. Enable Maintenance Mode to block users and admins
  7. View all bookings and users in respective sections

As Admin:

  1. Login with admin@club.com / admin123
  2. View analytics dashboard
  3. Navigate to Events and click "Add Event"
  4. Fill in event details: title, description, date, time, price, capacity, venue
  5. View all bookings in Bookings section
  6. Send notifications to all users from Notifications

As User:

  1. Login with user@club.com / user123
  2. Browse available events in Events section
  3. Click Book Now on any event
  4. Select number of guests
  5. Confirm booking and see total amount
  6. View booking history in My Bookings
  7. Check Notifications for updates

üé® Features Walkthrough

Event Management

  • Create events with title, description, date, time, price, capacity, and venue
  • Edit existing events
  • Delete events (with confirmation)
  • Track bookings per event
  • Automatic capacity management

Booking System

  • Users can book events with multiple guests
  • Real-time capacity checking
  • Automatic total calculation
  • Booking confirmation
  • Booking history for users
  • All bookings visible to admins and owners

Notification System

  • Admins can send notifications to all users
  • Users receive notifications in their dashboard
  • Timestamp for each notification
  • Persistent notification history

Maintenance Mode

  • Owner can enable/disable maintenance mode
  • Custom maintenance message
  • Blocks all users and admins
  • Owner can still access the system
  • Full-screen maintenance screen
